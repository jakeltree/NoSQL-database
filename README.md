# task3
Фуллстэк проект для предмета "Работа на ЭВМ" мехмата МГУ 2023.
Создается класс для хранения базы данных в формате JSON и используется клиентно/серверная архитектура для работы с ним.

# Собрать проект:
cd task3_build
cmake ../task3

# Запустить:
Сервер 
        cd task3_server
       ./main_server
Клиент
       cd task3_client
       ./main_client
